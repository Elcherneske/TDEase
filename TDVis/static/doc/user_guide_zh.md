## 用户指南

TDvis 是为 Topdown 数据设计的交互式可视化工具，
我们尝试以中国传统卷轴展开的方式来展现质谱的多维度数据

**核心价值**：
- 🎨 热力分布图谱：直观展示蛋白质变体在液相色谱中的分布状况
- 🔍 智能峰识别：自动计算质量间隔，辅助 PTMs 识别
- 🛠️ 自定义分析：支持用户定义修饰类型进行靶向检索
- 📊 谱图的直观展示与比对: 提供三维展示以及谱图对比的展示

## 快速指南

### 1. 数据准备
**文件要求**：
- 选择 Toppic 生成的原始结果文件夹
- 确保包含 `_html` 子目录（网页版通过界面直接选择即可）

![数据选择界面](../image/FileSelect.png)

### 2. 报告总览
**功能入口**：首个标签页展示基础检测报告  
**核心信息**：
- 蛋白质检出总量
- ms feature表格的呈现与下载

![主报告界面](../image/MainReport.png)

### 3. 三维可视化分析

#### 3.1 特征热力图
**坐标系**：
- X 轴：色谱保留时间（分钟）
- Y 轴：碎片质量（Da）

![热力图示例](../image/FeatureMap.png)

**操作指南**：
1. 使用工具栏第四个「框选工具」选取目标区域
   - *可以通过双击来退出原本的框选状态。*
2. 程序自动生成选定区域的积分谱图
3. 典型案例（Histone 分析）：
   ![alt text](../image/histone.png)
   在其中可以看到五个密集的峰群，正对应对应着Histone蛋白的五种主要形态。

#### 3.2 积分谱图分析
**坐标系**：
- X 轴：碎片质量（Da）
- Y 轴：归一化强度（%）

![积分谱图](../image/spectrum.png)

**分析技巧**：
- 定位最高强度峰作为基准
- 峰间距离：*设定目标质量与相邻峰之间的距离*

#### 3.3 PTMs 鉴定
**过滤参数**：

- 目标质量: 使用上一步中选中的最高强度的峰，当然也支持*手动输入假设质量（可以是图中不存在，但是是我们希望得到的）*
- 质量范围: *相邻峰搜索半径（±Da），程序会在这个范围内进行临近峰的检索与匹配，而非框选的区域*。
- 强度阈值:*过滤低信号峰（>设定值%）*，从而避免信息的冗余。如果您的目标峰找不多，或许是信号太低，被过滤掉了。

![参数面板](../image/NeighborFIlter.png)

**高级功能**：
1. **自定义修饰**：添加用户定义 PTMs 类型  
   ![自定义界面](../image/CustomPTMs.png)
2. **组合修饰**：分析多重修饰可能性  
   ![多修饰分析](../image/NeighborPeaks.png)
3. **批量输入参数**： 通过编写json格式的文件，您能够保存一套自己常用的参数，从而直接导入。

#### 3.4 更多形态的直观展示

1. **三维图像功能**: 实现三维的可视化
通过`热力图基本设置`中的`视图模式`来进行切换
![三维模式设置](../image/三维模式启动.png)

![三维图像](../image/三维图像.png)

2. **谱图对比**：支持多个谱图进行对比分析

这里我们以CYC的一条数据为例，其中有多次进样的信息。

首先点击启动该CheckBox
![对比展示设置](../image/compare_set.png)
然后借助于出现的 `选择报告文件夹` 进行文件选择(该文件夹就是最开始所选择的报告文件夹,如果选错了,就会报错)

接着选择您所需要使用的对比样本

当然,您还可以接着改变两个样本的展示颜色。

**在对比模式下，我们不再采用热力图的颜色模式**

- **二维模式**：不同的样品直接标注不同的颜色，不再渐变。
![compare_2D](../image/compare_2D.png)
- **三维模式**：两个样品也是标注了不同的颜色。不过我们设置了随强度而变化的透明度来达到更好的展示效果。
![compare_3D](../image/compare_3D.png)


- **镜像图**：*在2D模式下进行框选之后*，其所生成的积分谱图会呈现镜像的排布
   - ⚠ 需要注意的是，在该模式下，不支持进行PTMs的查询
![mirror](../image/mirror.png)

## 数据导出
1. 在第一个界面可以下载feature的tsv表格。
2. 在特征图像的界面，我们可以将感兴趣的图像导出为html格式可以交互的图像。
   - 均已给出了明确的按钮。
3. 在第三个界面，我们提供了Toppic软件所解析出的二级信息的文件。
